# Skyrim Alchemy Lab

A web-based alchemy workbench for **The Elder Scrolls V: Skyrim**. Browse all ingredients, drag them into a 3-slot workbench, and instantly see potion effects and estimated gold values. Smart suggestions help you pick the most valuable combinations.

## Features

- Complete ingredient database with English & Chinese names
- 3-ingredient potion workbench with real-time effect calculation
- Smart suggestions that recommend the most valuable next ingredient
- Sortable & searchable ingredient list
- Responsive design (desktop + mobile)

```bash
npm run dev
```

Open http://localhost:3000
